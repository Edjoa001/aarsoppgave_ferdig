Hvordan kjøre spill:

1. Besøk https://github.com/Edjoa001/aarsoppgave_ferdig
2. Trykk på grønn knapp som heter "<> Code" og dermed nederst trykk download ZIP
3. Finn filen din på pc-en din, høyre klikk > Extract all
4. Trykk deg inn på filen > index.html
5. Spill spillet
